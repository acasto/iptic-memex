Open links in browser                      %%OPENLINK%%
 + Opens URLs in the system's default browser
 + Automatically adds https:// if no protocol is specified
 + Works cross-platform (macOS, Linux, Windows)
 + The URL can be provided in the content block
 + Examples:
   - Open a website:
%%OPENLINK%%
https://github.com
%%END%%
   - Open with automatic https:// addition:
%%OPENLINK%%
google.com
%%END%%
   - Open multiple links:
%%OPENLINK%%
https://docs.anthropic.com
https://platform.openai.com/docs
%%END%%
